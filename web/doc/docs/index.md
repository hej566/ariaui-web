# Aria UI Web

Aria UI Web provides browser-native custom element packages under the `@ariaui-web` scope. Every package has a native component spec, package-level unit tests, and documentation for the custom element API.

## Package coverage

- 63 packages under `@ariaui-web`
- component packages expose separated custom-element part modules
- utility packages keep package-specific unit tests and native contract docs
